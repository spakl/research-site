size consistency and size extensivity of qm methods

A method is size consistent if the energy of two non-interacting systems equals the sum of the energies of the individual systems. A method is size extensive if it scales correctly (linearly) with system size. Several methods have already been shown mathematically to be size consistent, size extensive, or both. In this study, we would like to demonstrate that computationally as well.
